# chore-companion(プロトタイプ)

家事モチベーションアプリの、動く画面確認用プロトタイプ。設計の一次資料は
[`../docs/chore-motivation-app/`](../docs/chore-motivation-app/) を参照。

キャラの名前・見た目はまだ未定のため、吹き出しはアイコン(🐣)のみで表示している。

## 起動方法

```bash
npm install
npm run dev
```

`http://localhost:3000` を開く。データはブラウザの localStorage に保存され、日付が変わると
その日の状態(まだ/もうやった/今日はサボる)がリセットされる。

## 実装済みの範囲

- タイトル → キャラ(状態確認・時間帯でセリフのトーンが変わる) → ホーム
- ホーム(まだ)→ 家事表 / おみくじ
- 家事表:チェックで完了(繰り返し項目はチェック日から起算して次回期限を再計算)、
  「+ メモを追加」で一時的/繰り返す項目を追加、一時的項目は削除可能
- おみくじ:期限が来ている家事からランダムに提案。低確率でチャレンジ枠(マンネリ化対策)が混ざる
- おみくじ結果 → ホーム(ミッション保留中)→ 完了報告 でゲージ加点
- 「今日はサボる」→ 雑談(家事の話はしない)→「やっぱやる気出てきた」でおみくじへ復帰
- デフォルトの家事表(一人暮らし向けの繰り返し項目9個)とおみくじのチャレンジ候補(8個)を用意
- 進化:頑張りが貯まって次の進化ライン(7〜12回でランダム)に達すると、レベルアップ画面に遷移し
  ステージが進む。ゲージは0に戻り、次の進化ラインを引き直す
